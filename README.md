A mini project using **Python**, **NLP**, **simple ML scoring**, and **data analytics**

- Rule-based ML scoring
- Mood history tracking
- Automatic bar-graph plotting (matplotlib)
- Graph saved as PNG

## Technologies
- Python
- NLP
- Matplotlib
- ML-style scoring

## How to Run
1. Install matplotlib:
   pip install matplotlib
2. Run the file:
   python mood_music_plot_safe.py
